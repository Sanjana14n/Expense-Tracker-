# Expense-Tracker-
The Expense Tracker is a web-based application developed using Python (Flask) and SQLite that helps users manage their daily financial activities in a simple and organized way. The system allows users to create an account, log in securely, and record their expenses with details like title, amount, and date. It also enables users to set their monthly salary and automatically calculates total expenses, monthly spending, and remaining balance. The dashboard provides a clean and professional interface displaying key statistics such as total expenses, salary, balance, highest expense, and average daily spending. Users can edit or delete expenses, filter records by month, and visualize their spending patterns using interactive charts powered by Chart.js. The project focuses on combining financial logic with a modern user interface, making it both practical and user-friendly. Overall, it demonstrates full-stack development skills including authentication, database management, CRUD operations, financial calculations, and responsive UI design.
